== Description ==

Trendmag theme is designed for magazines and e-commerce with flexible layout. The theme is based on KOPATHEME layout manager technique that will let you flexibility choose layout options of every pages within your site. It is very helpful when you are experimenting with visual hierarchy. You can define unlimited sidebar for widget areas, and with powerful custom widgets, the theme provides you more flexibility and ease-of-use for your site.

== Installation ==

1. Upload the files to the /wp-content/themes/trendmag/ directory.
2. Activate the theme through the 'Themes' menu in WordPress.
3. You can install plugin trendmag-toolkit (https://wordpress.org/plugins/trendmag-toolkit/ ) to active some widgets and shortcodes for trendmag theme
